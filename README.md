# Acoustic Deterrent Analysis

Code and data for:

"Evaluating Acoustic Deterrents for Reducing Migratory Bird Collisions with Buildings"

## Contents
- `analysis.R` — main analysis script (ARU + collision Bayesian models)
- `ARU_spring_data.csv` — ARU experiment data (Spring 2023)
- `NW_collision_data.csv` — collision data (Fall 2023–2024)

## Requirements
- R (>= 4.3)
- Packages: brms, dplyr, tidyr, lubridate, bayesplot, ggplot2

## Reproducing analyses
Run `analysis.R`. All data are included locally.

## Notes
Playback amplitude in field deployments was approximately 70 dB measured at 50 m from the speaker.
